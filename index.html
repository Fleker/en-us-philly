EN-NJ / EN-PA

A NEW LOCALE PROPOSAL

This website serves as a proposal to introduce a new IETF dialect to the standard, en-NJ (or en-PA), to enable software developers to better provide services to users in New Jersey and Philadelphia.

The language in this region is unique, with a number of specific words and grammar. As such, standard American English (en-US) does not fully express the actual way that people talk. The goal of these IETF dialects is to better localize language as it is used in software, and this should include the jawns that we say.

If this proposal is accepted, this new locale should be added to platforms such as web browsers and mobile operating systems as a usable language. Then, applications will be able to include a new set of strings for people in this region.

The population of Pennsylvania and New Jersey combined is about 22 million people. This is the size of many small nations and serves as an indication for the impact this new locale would have.

Although Philly English is primarily based on American English, there are many words and phrases which are unique to the region. You can learn more in the exploration tool below.

Some words in this locale have become used enough that they have definitions on Google. This further provides justification for helping to keep supporting this culture.

You can read more about this locale in articles such as the Washington Post.

GLOSSARY

Below is a list of common Philadelphia vernacular. It is not a complete list.

  • jawn - Stuff, things, a collection
  • hoagie - A long sandwich on a hoagie roll
  • shore - The beach
  • fam - Your close friends
  • jimmies - Sprinkles
  • ratchet - Repugnant or disgusting
  • Wawa - A convenience store that provides coffee and hoagies
  • water ice - A cold drink with flavored syrups, like a snow cone
  • cheesesteak - A food consisting of beefsteak with onions and cheese whiz
  • witout - A cheesesteak without onions
  • boul - A young man
  • ard - Alright
  • Mummer - A colorfully dressed marching band

PHILLYSCRIPT

PhillyScript is a new programming language, built on top of JavaScript, to help bring the uniqueness of Philadelphia to programming. 

boul Animal {
    new(legs) {
        youse.legs := legs || 2;
    }
}

boul Dog <- Animal {
    new() {
        super(4);
    }
}

jawn* foo := new Animal(2);
jawn a := jawns(1,2,3,4);
if (a.witout(5)) {
    jawn* b := 'hello';
} else if (a.whereyat(1) === 0) {
    jawn* b := 'world';
}
        
if (foo.legs === 2) {
    :) Human
} else if (foo.legs !== 4) {
    :| Does not have 2 or 4 legs
}

Programming Basics

  • You don't declare variables with `let` or `const`. Instead you use `jawn` or `jawn*`. This way, there are fewer keywords to memorize.
  • Arrays can be constructed by using `jawns(1,2,3,...)`
  • Instead of using `this`, the keyword `youse` is used
  • The method `includes` is replaced by `wit`. Also, checking the inverse can be done using `witout`.
  • The method `indexOf` is replaced by `whereyat`.
  • Classes are defined with `boul` and are extended by calling `<-`

You can learn more about PhillyScript, including trying it out for yourself, on the PhillyScript page.
