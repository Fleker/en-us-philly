EN-US-PHILLY

A NEW LOCALE PROPOSAL

This website serves as a proposal to introduce a new RFC 5646 variant to the standard, en-US-philly, to enable software developers to better provide localized services to users in New Jersey and Philadelphia. While most of these localized services have traditionally been print media, the advent of voice assistants highlights the value of providing auditory responses that better reflect the local culture.

The language in this region is unique, with a number of specific words and grammar. As such, standard American English (en-US) does not fully express the actual way that people talk. The goal of these standardized dialects is to better localize language as it is used in online tools and media, and this should include the jawns that we say.

The population of Pennsylvania and New Jersey combined is about 22 million people. This is the size of many small nations and serves as an indication for the impact this new locale would have.

Looking up the specifications for RFC 5646, which defines the way that dialects are developed, as well as consulting the IANA Language Subtag Registry, it seems that this dialect would best fit as a new variant subtag that is based on the existing en-US locale.

As stated in section 3.6 of the RFC 5646, "variations within a language...or distinguishing variation MAY be registered as variant subtags". The contents of this website demonstrate the unique variations of English within Philadelphia, lending support to the idea that this should be considered a standardized variant.

This variant is being submitted as an entirely new subtag, as it is not defined as an ISO 639 code nor any code defined in other ISO specifications.

This website represents a formal intent to register this subtag, and the site is also is meant to collect support from the general public.

As precedent, there is already submitted the tag en-CA-newfound to represent the 500 thousand or so inhabitants of Newfoundland, on the eastern cost of Canada.

The specific vernacular of the Philadelphia region has been studied in detail by linguists including William Labov, from the University of Pennsylvania. He has written a number of books and academic works that highlight this variant of standard US English. Unlike a simple accent, the variation of Philadelphia English introduces new words into the language and has a number of grammatical differences.

Although Philly English is primarily based on US English, there are many words and phrases which are unique to the region. It was developed by a diverse group of inhabitants, creating a unique blend of African American Vernacular English along with influnces from European populations like the Irish, Italian, and German immigrants. You can learn more in the exploration tool below. This tool does not show specific variations in how words are said in Philadelphia compared to standard US English.

As defined in section 3.5 of RFC 5646, a variant subtag can be independently registered by sending an email to the IANA mailing list with the proposed record, as defined below.

Some words in this locale have become used enough that they have definitions on Google. This further provides justification for defining Philadelphia as a unique variant of US English.

You can read more about this locale in articles such as the Washington Post. You can learn more about RFC 5646 from W3C. You can find RFC 5646 tags through this handy tool. Finally, more references are available at the bottom of this page.

PROPOSED IANA RECORD

Type: variant
Subtag: philly
Description: Philadelphia dialect of US English
Added: 2018-xx-xx
Prefix: en-US

GLOSSARY

Below is a list of common Philadelphia vernacular. It is not a complete list.

  • jawn - Stuff, things, a collection
  • hoagie - A long sandwich on a hoagie roll
  • shore - The beach
  • fam - Your close friends
  • jimmies - Sprinkles
  • ratchet - Repugnant or disgusting
  • water ice - A cold drink with flavored syrups, like a snow cone
  • cheesesteak - A food consisting of beefsteak with onions and cheese whiz
  • witout - A cheesesteak without onions
  • boul - A young man
  • ard - Alright

PHILLYSCRIPT

PhillyScript is a new programming language, built on top of JavaScript, to help bring the uniqueness of Philadelphia to programming. 

boul Animal {
    new(legs) {
        youse.legs := legs || 2;
    }
}

boul Dog <- Animal {
    new() {
        super(4);
    }
}

jawn* foo := new Animal(2);
jawn a := jawns(1,2,3,4);
if (a.witout(5)) {
    jawn* b := 'hello';
} else if (a.whereyat(1) === 0) {
    jawn* b := 'world';
}
        
if (foo.legs === 2) {
    :) Human
} else if (foo.legs !== 4) {
    :| Does not have 2 or 4 legs
}

You can learn more about PhillyScript, including trying it out for yourself, on the PhillyScript page.
